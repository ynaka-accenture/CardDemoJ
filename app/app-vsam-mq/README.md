# VSAM-MQ オプション(日本版CardDemo)

口座照会のMQ連携を日本語化したオプションアプリケーション。

## 日本語化の焦点: MQ CCSID境界

口座に紐づく顧客の漢字氏名を MQ メッセージ本文へ組み立てて送出する。
MQ MD(メッセージ記述子)の CodedCharSetId(CCSID)が EBCDIC日本語(930)の
まま送信されると、受信側が UTF-8(1208)基盤の場合に漢字が文字化けする。
SO/SI を含む混在データの扱いはキューマネージャ設定に依存する。

## プログラム
- COACCJ01 — 口座MQ連携(英字名・実行可能)
- COACMJ01 — 口座MQ連携(日本語データ名版、MQMD CCSID明示)
- CODATJ01 — 日付処理
