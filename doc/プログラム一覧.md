# 日本版CardDemo 完全版 — プログラム一覧(全資産)

AWS CardDemo の全資産を日本語化した完全版の構成。

## 構成サマリ

| 区分 | 本数 | 内容 |
|------|------|------|
| COBOLプログラム | 42本 | 英字名40本(GnuCOBOL実行可・構文チェック合格)+ 日本語名2本(Enterprise COBOL DBCS利用者語) |
| BMS画面 | 15枚 | 全オンライン画面(DBCS/SOSI属性) |
| コピー句 | 15本 | 英字名9 + 日本語名6(レコード定義) |
| JCL | 9本 | 全バッチジョブ(日本語コメント) |
| IMS定義 | 2本 | DBD(漢字セグメント)+ PSB |
| MQ連携 | 1本 | CCSID境界を扱う口座MQ(日本語データ名) |

## 実行確認

`app/実行.sh` で英字名版の全バッチ(5サブアプリ)が稼働し、各トラップの
検出結果を表示する。全40本の英字名プログラムは GnuCOBOL 構文チェック合格。
日本語データ名版(顧客帳票/漢字解析 + 日本語コピー句)は IBM Enterprise COBOL
for z/OS の DBCS 利用者語を用いた本番ソースである。
